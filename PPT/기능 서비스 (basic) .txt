	- 로고


<로그인 페이지>
- 아이디 / 비밀번호 입력란
- 로그인 버튼
	- 있는 데이터인지 판단
- 아이디 찾기 	
	- 이름
	- 전화번호
- 비밀번호 찾기
	- 데이터에 아이디 있는지 판단
		- 전화번호 인증
- 회원 가입
	- 이름
	- 아이디
	- 닉네임
	- 성별
	- 사진(카카오톡
	- mbti
	- 다음-> 다음-> 회원 가입 버튼

<메인 페이지>
- 프로필 버튼 (profile)
	- 편집 기능 버튼
	- 닉네임
	- mbti
	- 관심사
	- 취미
	- 매너점수
		- 댓글
	- 사진 (프로필 한장)
	- 자기 소개
	- 회원 탈퇴 버튼 누르면 회원 정보 삭제
- message 버튼
	- 채팅 기능
	- 단체 톡방
	- 톡방 리스트
- 관심사를 통한 모임 페이지
	- API 따오고 조건 걸어서 경기도 수원시 권선구 구단위
- 원데이 모임 버튼
	- 날짜를 정하고 만나는 원데이 모임 글 목록
	- 회원가입할 때 등록한 관심사 키워드를 눌러 키워드가 등록된 모임만 보기 기능
- 정기 모임 버튼
	- 정기적으로 운영하는 정기 모임 글 목록
	- 회원가입할 때 등록한 관심사 키워드를 눌러 키워드가 등록된 모임만 보기 기능
- 내 모임 버튼
	- 내가 가입한 모임 보기
	- 모임 누르면 모임 채팅방으로 이동
